# 财经新闻 Skill

获取实时财经新闻和资讯。

## 适用场景

✅ **使用此 skill 当:**

- 询问今天的新闻
- 查询财经资讯
- 获取股市相关新闻
- 了解宏观经济动态

## 支持的新闻类型

- **财经快讯**: A股市场实时新闻
- **重要新闻**: 重磅财经新闻
- **宏观要闻**: 宏观经济政策新闻
- **全球财经**: 国际财经动态

## 使用方法

直接在对话中发送相关关键词:

| 关键词 | 说明 |
|--------|------|
| 新闻 | 财经快讯 |
| 重要新闻 | 重要新闻 |
| 宏观 | 宏观要闻 |
| 全球 | 全球财经 |

## 示例

```
新闻
财经新闻
今日财经资讯
```

## 数据来源

- 东方财富网
- 新浪财经
